# Task 01 — Project Scaffold

## Goal
Set up the Composer package structure, PSR-4 autoloading, Slim 4 bootstrap, and sample entry point so the app boots and responds to requests.

## Files to Create

```
composer.json
package.json
tailwind.config.js
assets/input.css
assets/styles.css        (generated by Tailwind CLI)
src/App.php
public/.htaccess
public/index.php
```

## Steps

1. **`composer.json`** — Define the package:
   - Name: `iserter/easy-lead-capture`
   - PSR-4 autoload: `Iserter\\EasyLeadCapture\\` → `src/`
   - Require: `slim/slim` ^4.0, `slim/psr7`, `php` ^8.1
   - Require-dev: `phpunit/phpunit` ^10
   - Add `symfony/mailer` and `google/recaptcha` as dependencies now (used in later tasks).

2. **`src/App.php`** — Main entry class:
   - Constructor accepts the config array, stores it.
   - `run()` method: creates a `Slim\App`, sets `base_path` from config, registers all route definitions (placeholder controllers for now), and calls `$slimApp->run()`.
   - Route registration should be in a private `registerRoutes()` method for readability.
   - Register routes: `GET /form`, `POST /submit`, `GET /embed.js`, `GET /admin`, `GET /admin/login`, `POST /admin/login`, `POST /admin/logout`, `GET /admin/export`.

3. **`public/index.php`** — Sample entry point:
   - Requires Composer autoload.
   - Creates a minimal config array (`base_path`, `admin.password`).
   - Instantiates `App` and calls `run()`.

4. **`public/.htaccess`** — Apache rewrite rules:
   ```apache
   RewriteEngine On
   RewriteCond %{REQUEST_FILENAME} !-f
   RewriteCond %{REQUEST_FILENAME} !-d
   RewriteRule ^ index.php [QSA,L]
   ```

5. **`package.json`** — Dev-only, for Tailwind CSS build:
   - `devDependencies`: `tailwindcss` ^3
   - Scripts:
     - `build:css`: `npx tailwindcss -i assets/input.css -o assets/styles.css --minify`
     - `watch:css`: `npx tailwindcss -i assets/input.css -o assets/styles.css --watch`

6. **`tailwind.config.js`**:
   - `content`: `['./src/Views/**/*.php']`
   - Extend theme as needed for the default color palette.

7. **`assets/input.css`** — Tailwind source:
   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;
   ```

8. **Run `npm install && npm run build:css`** to generate initial `assets/styles.css`.

9. **Run `composer install`** and verify the app boots without errors.

10. **Add to `.gitignore`**: `node_modules/`, `data/`. Do NOT ignore `assets/styles.css` — it must be committed.

## Acceptance Criteria
- `composer install` succeeds.
- `npm run build:css` produces `assets/styles.css`.
- Hitting `GET /form` returns a 200 (even if body is empty/placeholder).
- PSR-4 autoloading works — `Iserter\EasyLeadCapture\App` resolves correctly.
- `node_modules/` is gitignored; `assets/styles.css` is committed.
